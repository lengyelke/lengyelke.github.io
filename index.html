What is Activity Setting Report?

Activity Settings is a collection of report sources and entities for Moodle's built-in Custom Report feature. It helps course admins create detailed reports about activities and their settings, making course management and auditing easier than ever.

  • ✓ Adds new report sources for activities
  • ✓ Enables reporting on activity settings
  • ✓ Seamless integration with Custom Reports
  • ✓ Easy to install and configure

Example Use Cases

Audit Assignment Settings

Quickly generate a report listing all assignments in a course or category, including their due dates, submission types, and grading methods.

  • Columns: Course or category name, Assignment name, Due date, Submission type, Grading method
  • Conditions: Select course
Find Hidden Quizzes

Identify all quizzes that are currently hidden from students, helping ensure nothing is missed during course reviews.

  • Columns: Quiz name, Visible (under Course Module), Open/Close dates
  • Conditions:Visible = No
Compare Forum Settings

Compare all forums in a category to check subscription modes, group settings, and availability.

  • Columns: Forum name, Subscription mode, Group mode, Availability
  • Conditions: Select category
